## Build Setup

```bash
# clone project
$ git clone https://github.com/kasim393/link-previewer.git

# install dependencies and localhost for frontend
$ cd frontend
$ npm install
$ npm run dev

# install dependencies and locacalhost forbackend
$ cd backend
$ npm install
$ npm start
```
